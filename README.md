## Getting Started
1. install Tensorflow using Conda
2. install remaining python packages
```
$ pip install -r requirements.txt
```

### Training
```
$ python main.py -c configs/example.json
```

### Testing
```
$ python main.py -c configs/example.json --mode test
```
